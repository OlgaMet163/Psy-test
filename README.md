# Psy-test Bot

## О проекте
Телеграм-бот, который позволяет проходить психологические тесты (HEXACO, Ценностный опросник Шварца, Under Stress) и получать интерпретируемые результаты. На их основе пользователи смогут находить мэтчи — людей с совместимыми или, наоборот, комплементарными профилями.

## Установка и запуск
1. Создайте и активируйте виртуальное окружение Python 3.11+.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Скопируйте `config/env.template` в `.env` и заполните значения (`TELEGRAM_BOT_TOKEN`, опционально `DATA_DIR` и `DATABASE_PATH` для хранения SQLite).
4. Запустите бота:
   ```bash
   python -m bot.main
   ```

## Тестирование Hexaco
1. В Telegram откройте бота и отправьте `/start`.
2. Нажмите «Начать HEXACO» и ответьте на 24 утверждения (кнопки от «Точно не про меня» до «Точно про меня»).
3. По завершении бот выведет проценты и интерпретации по шести шкалам, а ответы и агрегированные данные сохранятся в локальной БД `data/bot.db`.
4. Для выгрузки CSV выполните:
   ```bash
   python scripts/export_hexaco.py --output exports
   ```
