# Тест HEXACO

## Назначение
Шестифакторная модель характера (HEXACO) оценивает личность по шести измерениям (Honesty-Humility, Emotionality, eXtraversion, Agreeableness, Conscientiousness, Openness). В боте тест используется для формирования цифрового профиля, рекомендаций и мэтчинга. Дополнительно считаются домены тёмной триады (макиавеллизм, нарциссизм, психопатия) — они нужны админу для внутренних аналитик.

## Формат прохождения
- Пользователь получает поочерёдно утверждения и отвечает с помощью кнопок.
- Состояние прохождения сохраняется после каждого ответа.
- После завершения расчёт шкал и краткая интерпретация отображаются сразу в интерфейсе; расширенный отчёт доступен админу.

## Варианты ответов и баллы
| Кнопка                     | Балл |
|---------------------------|------|
| Точно не про меня         | -2   |
| Скорее не про меня        | -1   |
| Частично про меня         | 0    |
| Скорее про меня           | +1   |
| Точно про меня            | +2   |

## Диапазоны процентов
- `[0; 13]` — Очень низкий
- `(13; 31]` — Низкий
- `(31; 68]` — Средний
- `(68; 87]` — Высокий
- `(87; 100]` — Очень высокий

## Алгоритм расчёта
1. Каждому утверждению соответствует числовой ответ из диапазона `[-2; 2]`.
2. Для утверждений с отрицательным вкладом знак ответа инвертируется (например, `+2` превращается в `-2`).
3. Суммируем все значения внутри домена: `S = Σ adjusted_value`.
4. Определяем диапазон домена: при `N` утверждениях ширина диапазона равна `4N`, а минимальный балл `-2N`.
5. Переводим сумму в процент:  
   `percent = ((S + 2N) / (4N)) * 100`.
6. Округляем до целых и присваиваем интервалу согласно шкале выше.

### Пример
У домена «Экстраверсия» три утверждения, значит `N = 3`, диапазон `[-6; 6]`, ширина `12`. Если сумма ответов после инверсий равна `2`, процент = `((2 + 6) / 12) * 100 = 66,7%`, что попадает в интервал «Средний».

- Пользователь получает результаты и интерпретации по шести классическим шкалам HEXACO: Экстраверсия, Добросовестность, Доброжелательность, Нейротизм, Открытость опыту, Честность-скромность.
