# Откат к английской версии

Резерв с текущей английской конфигурацией: `backup_en_full_2025-12-09` (создан перед русификацией).

Полный откат:
```bash
# 1) удалить русифицированную версию
rm -rf bot
# 2) вернуть английскую копию
cp -R backup_en_full_2025-12-09 bot
# 3) опционально проверить клавиатуру
PYTHONPATH="$(pwd)" pytest tests/test_keyboards.py
```
